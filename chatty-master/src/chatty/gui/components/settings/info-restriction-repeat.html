This can restrict which messages are checked for repetition. Once a message has passed this test, it will be compared against previous messages of the same user (even ones that would not pass this test, if you match on the message text).

Examples:
  • !status:M - Don't check messages of mods
  • !status:Mv - Don't check messages of mods or VIPs
  • mystatus:M - Only in channels where you are mod
You can also combine prefixes: !status:M mystatus:M
