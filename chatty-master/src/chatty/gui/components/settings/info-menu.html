[help-commands:commands-list Format 1]: Add several names from the [help-commands: Custom Command] list or timeout actions per line (no parameters):
/Command1 /Command2 30s 10m 6h 1d
/Command1 | /Command2 (vertical bar) to add [help-commands:separators separator] between items.
[help-commands:inline-commands Format 2]: Define a Custom Command directly (without adding it to the Custom Commands list), one per line:
Slap=/me slaps $$1 around a bit with a large trout
Buttons in different [help-commands:submenus rows]:
Put @<row name> on it's own line and prepend . on following lines.
@a1 (default row), @b1 (default bottom row, also via //Command1), anything starting with @a is another top row, bottom row otherwise
@a2
./Command1 /Command2
.Spoiler=/timeout $$1 600 no spoilers
[help-commands:shortcuts Shortcuts] which can be triggered when the User Dialog has focus:
/Ban[B] /Unban[U] 5s[1] 10m[2] 1d[3]
Slap[B]=/me slaps $$1 around a bit with a large trout
Add custom [help-commands:submenus submenus]:
@<menu name> on it's own line and prepend . on following lines.
@Rules
./No_Spam /No_Spoilers
.Spoiler=/timeout $$1 600 no spoilers
Parameters (User Dialog):
$1 Name of user, $2- Ban reason (if selected) ([help-commands:parameters-context more..])
Parameters (Streams Context Menu):
$1- Names of the selected streams ([help-commands:parameters-context more..])
Parameters (Channel Context Menu):
$1 Name of currently active channel (without leading #) ([help-commands:parameters-context more..])
[help-commands:replacements Parameters] (User Context Menu):
$1 Name of the user ([help-commands:parameters-context more..])
Parameters (Selected Text Context Menu):
$1- or $(msg) Selected text ([help-commands:parameters-context more..])
Parameters (Admin Context Menu):
$(title), $(game) ([help-commands:parameters-context more..])
