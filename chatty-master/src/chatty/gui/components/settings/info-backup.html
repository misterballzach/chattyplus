Backups are a copy of the settings which can be loaded if there is an issue with the regular settings file or to revert settings to an older version. Backups are created automatically, but can also be created manually via "Main - Save..".

  1. Select a backup from the list
  2. Click the "Load Selected Backup" button to overwrite the settings file with the backup
  3. Restart Chatty when you're done to load the new settings

Note: Once you loaded a backup Chatty will no longer automatically save your settings until you restart.
