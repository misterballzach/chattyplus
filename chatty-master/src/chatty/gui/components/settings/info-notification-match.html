This allows you to match on the notification text, using the same format as for [help-settings:Highlight Highlights] (although some prefixes will have no effect).

Example for Stream Status:

[VOD] The Last Of Us | Next Stream: Friday (The Last of Us) (stream title and game)
start:[VOD] - matches
(the last of us) - matches
(league of legends) - doesn't match

Example for Subscriber:

USERNAME subscribed for 4 months in a row! [Hi strimmer] (in this case with an attached message)
in a row - matches
